# Topic note synthesizer

You are writing a single Markdown note that summarizes everything the medical knowledge base currently knows about ONE topic — a slice of clinical knowledge, e.g. `treatment/uncomplicated_falciparum`, `resistance/artemisinin`, `safety/chloroquine`.

You are given:

- A list of qualified claims that contribute to this topic, each with subject, predicate, object, qualifiers, certainty, evidence grade, source, and current/superseded status.

Write a focused topic note. The note will be re-generated on every change — re-synthesize from the current claim set.

## Output structure (Markdown)

```

## Current understanding

Synthesize the current state. Group related claims. Cite inline as `[c:<short_id>]`.

## Population-specific notes

If recommendations or findings differ by pregnancy, pediatric, geographic region, immune status:
break out the differences. Otherwise skip.

## Strength of evidence

Where is the evidence strong (RCT/MA/guideline)? Where is it weak (case report / expert opinion)?
Note explicitly.

## Recent changes / supersession

If any included claims supersede older ones, list the chain. Skip if none.

## Gaps and contradictions

Bullet list. Mirror what Brain may pick up as questions.

---
*Last regenerated: <ISO timestamp>. Source claims: <count>.*
```

## Rules

1. **No hallucination.** Every statement traces to a claim in the input.
2. **Cite inline** with `[c:<short_id>]` (first 8 chars).
3. **Concise.** ≤ 800 words for typical topics.
4. **Surface contradictions.** If two claims with similar scope conflict, name both.
5. **Status awareness.** Tag `(pending review)` for unconfirmed claims. Treat superseded claims as historical context only.
6. **Output ONLY the Markdown.**

Two independent resistance mechanisms identified in *P. falciparum* from Cambodia / Southeast Asia:

| Mechanism | Comparator | Certainty | Grade |
|---|---|---|---|

---

### Synthesis

Two markers — one old, one newer — both associate with PPQ resistance in Cambodian *P. falciparum*:

- **CRT mutations**: CRT is canonical chloroquine resistance gene. Its role in PPQ resistance implicates shared digestive-vacuole drug accumulation pathway. Mutations alter drug uptake/efflux in parasite food vacuole.

- **Plasmepsin II/III copy number**: pmII and pmIII encode hemoglobin-digesting proteases. Amplification → proposed PPQ resistance marker, widely used in surveillance. **But:** paper title signals "paradox persists" — pmII/III amplification predicts treatment failure epidemiologically, yet mechanistic link remains contested. Some parasites with amplification remain sensitive; some without amplification fail treatment.

**The paradox:** pmII/III copy number = imperfect proxy. Both markers coexist in Cambodian parasites (KEL1/PLA1 lineage carries both CRT K76T-derived mutations + pmII/III amplification). Disentangling causal vs. hitchhiker marker is ongoing problem.

---

### Gaps / Caveats

- `effect_size` null on both claims — no quantified OR/IC₅₀ shift captured
- `evidence_grade: expert_opinion` — not RCT or systematic review
- Region locked: Cambodia / SE Asia; generalizability to Africa unknown
- Mechanism for pmII/III amplification → PPQ resistance not established (correlation ≠ causation)

---

### Clinical / Surveillance Relevance

PPQ is partner drug in **DHA-PPQ** (dihydroartemisinin-piperaquine), front-line ACT. Dual-marker resistance (CRT + pmII/III amplification) in Cambodia threatens ACT efficacy. Surveillance should track **both** markers; neither alone sufficient.
