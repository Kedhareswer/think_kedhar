MedBrain · Agents & Plan

Perpetually-learning medical knowledge base. Four agents on independent loops, communicating through SQL and a markdown corpus. SQL is source of truth; everything else is a derived view.
① active learning — Brain → questions.md → Student ② compaction — corpus ⇄ Dream → derivative + archive ③ retrieval — corpus → Graphify → CDS
